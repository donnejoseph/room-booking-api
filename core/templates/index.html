Meeting Room Booking API

A REST API for booking meeting rooms in an office environment

API Documentation

Explore and interact with the API using one of the following documentation interfaces:

Swagger UI ReDoc

Key Features

  • User authentication with JWT tokens
  • Room management (admin only for creation/modification)
  • Booking management
  • Room availability checking
  • API documentation with Swagger/ReDoc

Quick Start

Authentication

To use the API, you'll need to authenticate first:

POST /api/v1/auth/token/
{
  "username": "your_username",
  "password": "your_password"
}

Include the returned token in your subsequent requests:

Authorization: Bearer <your_access_token>

Common Endpoints

  • GET /api/v1/rooms/ - List all meeting rooms
  • GET /api/v1/bookings/ - List your bookings
  • POST /api/v1/bookings/ - Create a new booking
